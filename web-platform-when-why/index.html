## Native & Web
## What we know about Web development
Web apps… * Run across any OS with a browser * Are safer and are more isolated * Spread faster thanks to URLs * Show always up to date content * Are cheaper and reduce costs
The Web… * Is free (as in free-speech) * Has no friction * It is ubiquitous

## What the Web is missing
Web apps… * Are only accesible through the browser * Depends on the browser to be running * Can not access basic engagement features * Display non native UI that feels unnatural * Break if there is no connectivity
## Web APIs at the rescue!
* Service Workers API * Web Notifications * Push API * Web Manifest * Add-to-home-screen prompt
## The right choice
* Does your product need mobile APIs? * How much budget do you have? * How heterogeneus is your audience? * Do you need desktop presence? * What kind of content do you offer? * How fast is your release cycle?
